
## Getting Started
After cloning project from git hub 

##install required packages
run npm install from the root dir /employeeapp 

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## General info
I kept the site very simple i did not add any extra packages. per the requirements all components 
are server components.  I used scss with css nesting for my styling.  I used standard nextjs file 
based routing with a dynamic route for the person detail page.
